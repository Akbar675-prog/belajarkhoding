Tailwind CSS

Framework CSS berbasis utility-first untuk membuat desain cepat dan konsisten langsung di HTML. Pelajari class, layout, dan customisasi Tailwind.

Apa itu Tailwind CSS?

Tailwind CSS adalah framework CSS utility-first yang memungkinkan Anda membangun desain kustom langsung di markup HTML Anda, tanpa harus menulis CSS kustom.

Berbeda dengan framework seperti Bootstrap yang menyediakan komponen siap pakai, Tailwind memberikan kelas utilitas tingkat rendah yang dapat Anda gabungkan untuk membuat desain yang unik.

Keunggulan Tailwind CSS:

  • Utility-first: Ribuan kelas utilitas untuk styling cepat
  • Responsif: Prefix seperti md:, lg: untuk desain responsif
  • Customizable: Mudah dikonfigurasi melalui file konfigurasi
  • Produktif: Mempercepat proses development UI
  • Ringan: Hanya menghasilkan CSS yang Anda gunakan

Instalasi Tailwind CSS

1. Menggunakan CDN (Cara Paling Cepat)

Tambahkan script Tailwind ke HTML Anda:

<!-- CSS Tailwind via CDN -->
<script src="https://cdn.tailwindcss.com"></script>

Catatan: CDN tidak direkomendasikan untuk produksi karena ukuran file besar dan tidak ada customisasi.

2. Instalasi via npm (Direkomendasikan)

npm install -D tailwindcss
npx tailwindcss init

Kemudian buat file CSS utama Anda dan tambahkan:

@tailwind base;
@tailwind components;
@tailwind utilities;

3. Menggunakan CLI

Tailwind menyediakan CLI untuk build CSS:

npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch

Utility Class Tailwind

1. Spacing (Padding & Margin)

Tailwind menggunakan sistem spacing yang konsisten:

Padding semua sisi (p-4)
Padding horizontal & vertikal (px-4 py-2)
Margin kiri 8 unit (ml-8)
<!-- Contoh spacing -->
<div class="p-4">Padding semua sisi</div>
<div class="px-4 py-2">Padding x & y</div>
<div class="ml-8">Margin kiri</div>

2. Warna

Tailwind menyediakan palet warna yang luas:

Sukses Error
<!-- Contoh warna -->
<div class="bg-purple-500"></div>
<button class="bg-purple-600 text-white">Tombol</button>
<span class="bg-green-100 text-green-800">Sukses</span>

3. Typography

Tailwind menyediakan utility untuk styling teks:

Heading Besar

Subheading

Ini adalah paragraf dengan line-height yang nyaman dibaca. Tailwind membuat styling teks menjadi sangat mudah.

Teks kecil untuk keterangan

<!-- Contoh typography -->
<h1 class="text-4xl font-bold">Heading</h1>
<h2 class="text-2xl font-semibold text-purple-600">Subheading</h2>
<p class="text-gray-600 leading-relaxed">Paragraf</p>

Layout dengan Flex dan Grid

1. Flexbox

Tailwind menyediakan utility lengkap untuk flexbox:

1
2
3
<!-- Contoh flexbox -->
<div class="flex flex-wrap justify-between items-center">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

2. CSS Grid

Tailwind juga mendukung CSS Grid:

Kolom 1
Kolom 2
Kolom 3
<!-- Contoh grid -->
<div class="grid grid-cols-1 md:grid-cols-3 gap-4">
  <div>Kolom 1</div>
  <div>Kolom 2</div>
  <div>Kolom 3</div>
</div>

Customisasi Tema

Anda dapat dengan mudah menyesuaikan Tailwind dengan proyek Anda melalui file tailwind.config.js.

Contoh Konfigurasi Dasar

// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#7c3aed',
        secondary: '#f43f5e'
      },
      fontFamily: {
        sans: ['Inter', 'sans-serif']
      }
    }
  },
  plugins: []
}

Setelah mengubah konfigurasi, jalankan build process untuk menerapkan perubahan.

Contoh Kode Tailwind

Berikut contoh halaman HTML sederhana menggunakan Tailwind CSS:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Contoh Tailwind</title>
  <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-50">
  <div class="container mx-auto px-4 py-8">
    <h1 class="text-3xl font-bold text-center text-purple-600 mb-6">Selamat Datang</h1>
    <div class="grid grid-cols-1 md:grid-cols-2 gap-6">
      <div class="bg-white p-6 rounded-lg shadow">
        <h2 class="text-xl font-semibold mb-3">Card Pertama</h2>
        <p class="text-gray-600">Ini adalah contoh card dengan Tailwind CSS.</p>
        <button class="mt-4 px-4 py-2 bg-purple-600 text-white rounded hover:bg-purple-700">Tombol</button>
      </div>
      <!-- Card lainnya -->
    </div>
  </div>
</body>
</html>

Tips Menggunakan Tailwind

  • Gunakan ekstensi VS Code seperti "Tailwind CSS IntelliSense" untuk autocomplete class
  • Pelajari utility class dasar terlebih dahulu (padding, margin, warna, typography)
  • Gunakan prefix responsif (sm:, md:, lg:, xl:, 2xl:) untuk desain yang adaptif
  • Customisasi tema melalui tailwind.config.js untuk konsistensi desain
  • Gunakan @apply di CSS untuk membuat komponen yang sering digunakan
  • Jangan ragu menggabungkan beberapa utility class untuk efek yang diinginkan
  • Gunakan dokumentasi Tailwind sebagai referensi utama

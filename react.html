React.js

Library JavaScript untuk membangun antarmuka pengguna (UI) interaktif. Pelajari komponen, props, state, dan JSX.

Apa itu React.js?

React.js adalah library JavaScript open-source yang dikembangkan oleh Facebook untuk membangun antarmuka pengguna (UI) yang interaktif dan efisien.

React menggunakan konsep komponen yang dapat digunakan kembali, membuat pengembangan UI menjadi lebih modular dan mudah dikelola.

Keunggulan React.js:

  • Komponen: Membangun UI dengan komponen yang dapat digunakan kembali
  • Virtual DOM: Meningkatkan performa dengan meminimalkan manipulasi DOM langsung
  • JSX: Sintaks seperti HTML yang memudahkan penulisan komponen
  • Unidirectional Data Flow: Aliran data satu arah yang mudah dipahami
  • Ekosistem Kaya: Banyak library pendukung seperti React Router, Redux, dll

Memulai dengan React

1. Menggunakan CDN (Untuk Belajar)

Tambahkan script React ke HTML Anda:

<!-- Load React -->
<script src="https://unpkg.com/react@18/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>

Catatan: CDN hanya untuk pembelajaran, tidak direkomendasikan untuk produksi.

2. Create React App (Direkomendasikan)

Buat proyek React baru dengan perintah:

npx create-react-app nama-aplikasi
cd nama-aplikasi
npm start

3. Menggunakan Vite (Alternatif Cepat)

Vite adalah build tool modern yang lebih cepat:

npm create vite@latest nama-aplikasi --template react
cd nama-aplikasi
npm install
npm run dev

Komponen React

Komponen adalah blok bangunan dasar dalam React. Ada dua jenis komponen:

Function component adalah cara modern menulis komponen React menggunakan fungsi JavaScript:

import React from 'react';

function Salam() {
  return <h1>Halo Dunia!</h1>;
}

export default Salam;

Class component adalah cara lama yang menggunakan kelas JavaScript (masih digunakan tetapi kurang populer sekarang):

import React from 'react';

class Salam extends React.Component {
  render() {
    return <h1>Halo Dunia!</h1>;
  }
}

export default Salam;

Function component lebih sederhana dan direkomendasikan untuk pemula. Dengan React Hooks, function component bisa memiliki state dan lifecycle seperti class component.

JSX

JSX adalah sintaks seperti HTML yang digunakan dalam React untuk mendeskripsikan tampilan UI. JSX bukan HTML atau string, melainkan sintaks khusus yang dikompilasi menjadi JavaScript.

const element = <h1 className="judul">Halo, dunia!</h1>;

Contoh di atas akan dikompilasi menjadi:

const element = React.createElement(
  'h1',
  { className: 'judul' },
  'Halo, dunia!'
);

Aturan penting JSX:

  • Gunakan className bukan class
  • Gunakan htmlFor bukan for
  • Nama komponen harus diawali huruf kapital
  • Hanya boleh mengembalikan satu elemen root
  • Gunakan { } untuk menyisipkan ekspresi JavaScript

Props dan State

Props

Props (properties) adalah cara untuk mengirim data dari komponen parent ke komponen child. Props bersifat read-only dan tidak bisa diubah oleh komponen yang menerimanya.

function Salam({ nama }) {
  return <h1>Halo, {nama}!</h1>;
}

function App() {
  return <Salam nama="Budi" />;
}

State

State adalah data yang dimiliki dan dikelola oleh komponen itu sendiri. State bisa berubah dan ketika berubah, komponen akan dirender ulang.

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Tambah
      </button>
    </div>
  );
}

Event Handler

React memiliki sistem event yang disebut SyntheticEvent, yang merupakan wrapper di sekitar event browser asli. Berikut beberapa event handler umum:

Anda mengetik:

// Contoh event handler dalam React
<button onClick={() => alert('Diklik!')}>
  Klik Saya
</button>

<input type="text" onChange={(e) => console.log(e.target.value)} />

<form onSubmit={(e) => {
  e.preventDefault();
  alert('Form submitted!');
}}>
  <!-- form fields -->
</form>

Contoh: Counter Sederhana

Berikut contoh lengkap komponen counter menggunakan React dengan useState:

Counter: 0

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  const tambah = () => setCount(count + 1);
  const kurang = () => setCount(count - 1);
  const reset = () => setCount(0);

  return (
    <div className="text-center">
      <h3 className="text-xl font-medium mb-4">
        Counter: {count}
      </h3>
      <div className="flex justify-center gap-4">
        <button className="btn-demo" onClick={kurang}>-1</button>
        <button className="btn-demo" onClick={tambah}>+1</button>
        <button className="btn-demo bg-red-100 text-red-800" onClick={reset}>
          Reset
        </button>
      </div>
    </div>
  );
}

Struktur Folder Proyek React

Berikut struktur folder umum proyek React yang dibuat dengan Create React App:

nama-proyek/
├── node_modules/ # Dependensi yang diinstall
├── public/ # File statis (HTML, gambar, dll)
│ ├── index.html # Template HTML utama
│ └── ...
├── src/ # Kode sumber aplikasi
│ ├── App.js # Komponen utama
│ ├── index.js # Entry point JavaScript
│ ├── components/ # Komponen-komponen
│ ├── pages/ # Halaman-halaman
│ ├── assets/ # Gambar, font, dll
│ └── ...
├── package.json # Konfigurasi proyek dan dependensi
└── ...

Tips struktur folder:

  • Kelompokkan komponen berdasarkan fitur, bukan jenis file
  • Gunakan folder components untuk komponen yang digunakan kembali
  • Gunakan folder pages untuk komponen yang mewakili halaman
  • Simpan style terkait dengan komponennya (CSS Modules atau styled-components)
  • Gunakan konvensi penamaan yang konsisten

Django

Framework backend berbasis Python yang kuat dan lengkap. Pelajari struktur proyek Django, routing (URLconf), views, dan template.

Apa itu Django?

Django adalah framework web Python tingkat tinggi yang mendorong pengembangan cepat dan desain bersih. Django mengikuti filosofi "batteries included" dengan menyediakan hampir semua yang dibutuhkan untuk membangun aplikasi web modern.

Keunggulan Django dibanding framework lainnya:

  • Admin otomatis: Antarmuka admin yang bisa dikustomisasi
  • ORM kuat: Bekerja dengan database tanpa menulis SQL
  • Sistem template: Template engine yang fleksibel
  • Keamanan: Perlindungan bawaan terhadap banyak serangan web
  • Skalabilitas: Digunakan oleh situs besar seperti Instagram
  • Dokumentasi lengkap: Salah satu dokumentasi terbaik di dunia open source

Kapan menggunakan Django?

  • Aplikasi web dengan database kompleks
  • Sistem manajemen konten (CMS)
  • Aplikasi e-commerce
  • Situs berita atau portal
  • API backend

Instalasi Django

Sebelum menginstal Django, pastikan Python sudah terinstal di sistem Anda. Django kompatibel dengan Python 3.6 atau yang lebih baru.

Langkah-langkah instalasi:

  1. Pastikan Python sudah terinstal:
    python --version
  2. Buat virtual environment (disarankan):
    python -m venv env
  3. Aktifkan virtual environment:
    # Windows
    env\Scripts\activate

    # Mac/Linux
    source env/bin/activate
  4. Instal Django menggunakan pip:
    pip install django
  5. Verifikasi instalasi:
    django-admin --version

Membuat Proyek Django

Setelah Django terinstal, Anda bisa membuat proyek baru dengan perintah berikut:

django-admin startproject namaprojek

Perintah ini akan membuat struktur folder dasar untuk proyek Django Anda.

Menjalankan Server Pengembangan:

Masuk ke folder proyek dan jalankan server pengembangan:

# Masuk ke folder proyek
cd namaprojek

# Jalankan server
python manage.py runserver

Server akan berjalan di http://127.0.0.1:8000

Routing dan Views

Django menggunakan sistem URL routing yang disebut URLconf. Setiap aplikasi bisa memiliki file urls.py sendiri.

# namaapp/views.py
from django.http import HttpResponse

def index(request):
  return HttpResponse("Halo, Django!")

def about(request):
  return HttpResponse("Tentang Kami")
# namaapp/urls.py
from django.urls import path
from . import views

urlpatterns = [
  path('', views.index, name='index'),
  path('about/', views.about, name='about'),
]
# namaprojek/urls.py
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
  path('admin/', admin.site.urls),
  path('', include('namaapp.urls')),
]

Parameter URL:

Anda bisa menangkap parameter dari URL:

# urls.py
path('produk/<int:produk_id>/', views.produk_detail, name='produk_detail')

# views.py
def produk_detail(request, produk_id):
  return HttpResponse(f"Detail produk {produk_id}")

Template Django

Django memiliki sistem template yang powerful untuk memisahkan tampilan dari logika bisnis.

Langkah menggunakan template:

  1. Buat folder templates di dalam aplikasi Anda
  2. Buat file template HTML (contoh: index.html)
  3. Konfigurasi template di settings.py
  4. Render template dari view
# namaapp/views.py
from django.shortcuts import render

def index(request):
  context = {
    'judul': 'Halaman Utama',
    'daftar_produk': ['Laptop', 'Smartphone', 'Tablet']
  }
  return render(request, 'index.html', context)
# namaapp/templates/index.html
<!DOCTYPE html>
<html>
<head>
  <title>{{ judul }}</title>
</head>
<body>
  <h1>{{ judul }}</h1>
  <ul>
    {% for produk in daftar_produk %}
    <li>{{ produk }}</li>
    {% endfor %}
  </ul>
</body>
</html>

Sintaks Template:

  • {{ variable }} - Menampilkan nilai variabel
  • {% tag %} - Logika template (for, if, block, etc)
  • {# comment #} - Komentar
  • {% extends "base.html" %} - Template inheritance
  • {% include "partial.html" %} - Include template lain

Database dan Model

Django menyertakan ORM (Object-Relational Mapper) yang memungkinkan Anda bekerja dengan database menggunakan kode Python, tanpa harus menulis SQL secara langsung.

# namaapp/models.py
from django.db import models

class Produk(models.Model):
  nama = models.CharField(max_length=100)
  deskripsi = models.TextField(blank=True)
  harga = models.DecimalField(max_digits=10, decimal_places=2)
  stok = models.IntegerField(default=0)
  tanggal_dibuat = models.DateTimeField(auto_now_add=True)

  def __str__(self):
    return self.nama

Migrasi Database:

Setelah membuat model, Anda perlu membuat dan menjalankan migrasi:

# Buat migrasi
python manage.py makemigrations

# Jalankan migrasi
python manage.py migrate
# Menggunakan model di views.py
from .models import Produk

def daftar_produk(request):
  produk = Produk.objects.all()
  return render(request, 'produk/list.html', {'produk': produk})

def tambah_produk(request):
  if request.method == 'POST':
    Produk.objects.create(
      nama=request.POST['nama'],
      harga=request.POST['harga']
    )
    return redirect('daftar_produk')
  return render(request, 'produk/tambah.html')

Admin Django:

Django menyediakan antarmuka admin otomatis. Untuk menggunakannya:

  1. Buat superuser:
    python manage.py createsuperuser
  2. Daftarkan model di admin.py:
    from .models import Produk
    admin.site.register(Produk)
  3. Akses admin di /admin

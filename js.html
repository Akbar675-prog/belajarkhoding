Pemrograman Web

JavaScript

Bahasa pemrograman untuk membuat website interaktif. Pelajari variabel, fungsi, event, dan manipulasi DOM.

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Dengan JavaScript, Anda dapat membuat konten yang dinamis, mengontrol multimedia, menganimasikan gambar, dan banyak hal lainnya.

JavaScript adalah salah satu dari tiga teknologi inti dalam pengembangan web, bersama dengan HTML (struktur) dan CSS (tampilan).

// Contoh kode JavaScript sederhana
console.log('Halo, Dunia!');

Cara Menambahkan JavaScript ke HTML

1. Inline JavaScript

JavaScript langsung ditulis dalam atribut event HTML:

<button onclick="alert('Tombol diklik!')"> Klik Saya </button>

2. Internal JavaScript

JavaScript ditulis dalam tag <script> di HTML:

<script>
  function tampilkanPesan() {
    alert('Halo dari JavaScript!');
  }
</script>

3. External JavaScript

JavaScript ditulis dalam file terpisah (.js) dan dihubungkan dengan tag <script>:

<script src="script.js"></script>

Isi file script.js:

console.log('File JavaScript eksternal dimuat!');

Variabel dalam JavaScript

Variabel digunakan untuk menyimpan data. JavaScript memiliki 3 cara mendeklarasikan variabel:

Keyword Contoh Penjelasan
var var nama = "Budi"; Variabel dengan scope function (sudah jarang digunakan)
let let umur = 20; Variabel dengan scope block, nilainya bisa diubah
const const PI = 3.14; Variabel dengan scope block, nilainya tetap (tidak bisa diubah)

Coba Variabel

let nama = 'Budi';
let umur = 20;
const PI = 3.14;

console.log(`Halo, nama saya ${nama}, umur ${umur}`);

Fungsi dalam JavaScript

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi dijalankan ketika "dipanggil".

1. Function Declaration

function sapa(nama) {
  return `Halo, ${nama}!`;
}

console.log(sapa('Ani')); // Output: Halo, Ani!

2. Function Expression

const tambah = function(a, b) {
  return a + b;
};

console.log(tambah(5, 3)); // Output: 8

Event dalam JavaScript

Event adalah tindakan atau kejadian yang terjadi pada halaman web, seperti klik mouse, input keyboard, dll. JavaScript dapat "mendengarkan" event ini dan meresponsnya.

Event Penjelasan
onclick Ketika elemen diklik
onmouseover Ketika mouse berada di atas elemen
onmouseout Ketika mouse meninggalkan elemen
onkeydown Ketika tombol keyboard ditekan
onload Ketika halaman selesai dimuat

Manipulasi DOM

DOM (Document Object Model) adalah representasi dari halaman web yang dapat dimanipulasi dengan JavaScript. Dengan DOM, kita bisa mengubah konten, struktur, dan style halaman web.

Method Penjelasan
getElementById() Memilih elemen berdasarkan ID
querySelector() Memilih elemen pertama yang cocok dengan selector CSS
innerHTML Mengubah konten HTML dalam elemen
style.property Mengubah style CSS elemen
addEventListener() Menambahkan event handler ke elemen

Coba Manipulasi DOM

Ini adalah contoh teks yang akan diubah.

// Mengubah teks dalam elemen
document.getElementById('demo-text').innerHTML = 'Teks baru!';

// Mengubah warna background
document.getElementById('color-box').style.backgroundColor = 'blue';

Contoh Kode JavaScript Sederhana

Berikut contoh file JavaScript sederhana yang bisa Anda salin dan coba:

// script.js - Contoh file JavaScript eksternal

// Deklarasi variabel
const nama = 'Budi';
let umur = 20;

// Fungsi untuk menampilkan pesan
function tampilkanPesan() {
  alert(`Halo ${nama}, kamu ${umur} tahun!`);
}

// Event listener untuk tombol
document.getElementById('tombol-saya').addEventListener('click', tampilkanPesan);

// Mengubah style elemen
document.querySelector('.kotak').style.backgroundColor = '#f59e0b';

Salin kode di atas ke file dengan ekstensi .js dan hubungkan ke HTML Anda untuk melihat hasilnya.

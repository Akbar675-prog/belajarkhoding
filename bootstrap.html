Bootstrap

Framework CSS populer untuk membuat website responsif dengan cepat. Pelajari grid system, komponen, dan utilitas Bootstrap.

Apa itu Bootstrap?

Bootstrap adalah framework CSS open-source yang menyediakan kumpulan alat untuk membangun antarmuka pengguna (UI) yang responsif dan mobile-first dengan cepat dan mudah.

Bootstrap dikembangkan oleh Twitter dan dirilis pertama kali pada tahun 2011. Sekarang menjadi salah satu framework front-end paling populer di dunia.

Keunggulan Bootstrap:

  • Sistem grid responsif yang powerful
  • Koleksi komponen UI siap pakai
  • Utilitas CSS yang lengkap
  • Konsistensi tampilan antar browser
  • Dokumentasi yang sangat baik

Cara Menggunakan Bootstrap

1. Menggunakan CDN (Cara Paling Cepat)

Tambahkan link CSS dan JS Bootstrap ke HTML Anda:

<!-- CSS Bootstrap -->
<link href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JS Bootstrap (dengan Popper) -->
<script src="https://cdn.jsdelivr.net/npm/[email protected]/dist/js/bootstrap.bundle.min.js"></script>

2. Instalasi via npm (Untuk Proyek Modern)

npm install [email protected]

Kemudian impor di file JavaScript Anda:

import 'bootstrap';

3. Unduh Secara Manual

Unduh file Bootstrap dari situs resmi dan tautkan file CSS dan JS lokal ke proyek Anda.

Sistem Grid Bootstrap

Sistem grid Bootstrap menggunakan serangkaian container, row, dan column untuk mengatur dan menyelaraskan konten. Sistem ini dibangun dengan flexbox dan sepenuhnya responsif.

Struktur Dasar Grid

<!-- Container -->
<div class="container">
  <!-- Row -->
  <div class="row">
    <!-- Columns -->
    <div class="col">Kolom 1</div>
    <div class="col">Kolom 2</div>
    <div class="col">Kolom 3</div>
  </div>
</div>

Contoh Grid Bootstrap

Kolom 1
Kolom 2
Kolom 3

Grid ini akan menyesuaikan secara otomatis di berbagai ukuran layar.

Grid Responsif

Anda bisa menentukan lebar kolom untuk breakpoint yang berbeda:

<div class="row">
  <!-- 12 kolom di mobile, 6 di tablet, 4 di desktop -->
  <div class="col-12 col-md-6 col-lg-4">Kolom 1</div>
  <div class="col-12 col-md-6 col-lg-4">Kolom 2</div>
  <div class="col-12 col-md-6 col-lg-4">Kolom 3</div>
</div>

Komponen Bootstrap

1. Tombol (Buttons)

<!-- Contoh kode tombol Bootstrap -->
<button class="btn btn-primary">Primary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>

2. Navbar

Brand
Home Features Pricing About
<!-- Contoh kode navbar Bootstrap -->
<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Brand</a>
    <!-- Menu items -->
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" href="#">Home</a>
        </li>
        <!-- Item lainnya -->
      </ul>
    </div>
  </div>
</nav>

3. Card

Judul Card

Ini adalah contoh card Bootstrap dengan gambar, judul, teks, dan tombol.

<!-- Contoh kode card Bootstrap -->
<div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Judul Card</h5>
    <p class="card-text">Deskripsi card.</p>
    <a href="#" class="btn btn-primary">Baca Selengkapnya</a>
  </div>
</div>

Utilitas Bootstrap

Bootstrap menyediakan banyak kelas utilitas untuk styling cepat tanpa menulis CSS kustom.

1. Margin dan Padding

Kelas Contoh Penjelasan
m-*, p-* m-3, p-2 Margin/Padding di semua sisi (0-5)
mx-*, px-* mx-auto, px-4 Margin/Padding horizontal
my-*, py-* my-2, py-3 Margin/Padding vertikal
mt-*, pt-* mt-5, pt-0 Margin/Padding top

2. Warna dan Background

primary secondary success danger warning info light dark
<!-- Contoh penggunaan warna -->
<div class="bg-primary text-white p-3">primary</div>
<div class="bg-success text-white p-3">success</div>
<div class="bg-danger text-white p-3">danger</div>

3. Text Utilities

Rata kiri (text-start)

Rata tengah (text-center)

Rata kanan (text-end)

Tebal (fw-bold)

Miring (fst-italic)

Garis bawah (text-decoration-underline)

Contoh Kode Bootstrap Sederhana

Berikut contoh halaman HTML sederhana menggunakan Bootstrap:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Contoh Bootstrap</title>
  <!-- CSS Bootstrap -->
  <link href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container my-5">
    <h1 class="text-center mb-4">Selamat Datang</h1>
    <div class="row">
      <div class="col-md-6">
        <div class="card">
          <div class="card-body">
            <h5 class="card-title">Judul Card</h5>
            <p class="card-text">Ini adalah contoh card Bootstrap.</p>
            <a href="#" class="btn btn-primary">Tombol</a>
          </div>
        </div>
      </div>
      <!-- Kolom lainnya -->
    </div>
  </div>
<!-- JS Bootstrap -->
<script src="https://cdn.jsdelivr.net/npm/[email protected]/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Salin kode di atas ke file HTML dan buka di browser untuk melihat hasilnya.

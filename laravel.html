Pemrograman Web

Laravel

Framework PHP modern untuk membangun aplikasi web yang cepat dan elegan. Pelajari struktur Laravel, routing, controller, dan blade template.

Apa itu Laravel?

Laravel adalah framework PHP open-source yang menggunakan pola MVC (Model-View-Controller). Laravel menyederhanakan pengembangan web dengan sintaks yang ekspresif dan alat bawaan yang powerful.

Keunggulan Laravel dibanding PHP biasa:

  • Routing yang elegan: Sistem routing yang mudah dipahami
  • Blade Templating: Template engine yang powerful
  • Eloquent ORM: Sistem database yang intuitif
  • Artisan CLI: Alat command line yang membantu pengembangan
  • Autentikasi bawaan: Sistem login/register siap pakai
  • Komunitas besar: Dokumentasi lengkap dan ekosistem package yang kaya

Kapan menggunakan Laravel?

  • Aplikasi web yang kompleks
  • Proyek yang membutuhkan struktur kode yang rapi
  • Aplikasi dengan autentikasi pengguna
  • Proyek yang membutuhkan fitur database relasional
  • Pengembangan API RESTful

Instalasi Laravel

Untuk mulai menggunakan Laravel, Anda perlu menginstal Composer terlebih dahulu, kemudian menggunakan Composer untuk menginstal Laravel.

Langkah-langkah instalasi:

  1. Download dan install Composer dari getcomposer.org
  2. Buka terminal/command prompt
  3. Jalankan perintah berikut untuk menginstal Laravel:
    composer global require laravel/installer
  4. Buat proyek Laravel baru:
    laravel new nama-proyek
    atau
    composer create-project laravel/laravel nama-proyek
  5. Masuk ke folder proyek:
    cd nama-proyek

Routing Dasar

Routing di Laravel didefinisikan di file routes/web.php. Routing menghubungkan URL dengan fungsi atau controller.

// routes/web.php
// Route dasar
Route::get('/', function () {
  return 'Halaman Utama';
});

// Route dengan parameter
Route::get('/user/{id}', function ($id) {
  return 'User ID: ' . $id;
});

// Route dengan parameter opsional
Route::get('/user/{name?}', function ($name = 'Guest') {
  return 'Halo, ' . $name;
});

// Route dengan controller
Route::get('/profile', 'ProfileController@index');

Jenis-jenis Route:

  • Route::get() - Menangani request GET
  • Route::post() - Menangani request POST
  • Route::put() - Menangani request PUT
  • Route::patch() - Menangani request PATCH
  • Route::delete() - Menangani request DELETE
  • Route::any() - Menangani semua jenis request

Controller dan View

Controller bertanggung jawab untuk menangani logika aplikasi, sedangkan View bertanggung jawab untuk menampilkan tampilan.

// Membuat controller baru
php artisan make:controller NamaController
// app/Http/Controllers/UserController.php
<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class UserController extends Controller
{
  public function index()
  {
    $users = ['Budi', 'Ani', 'Citra'];
    return view('users.index', compact('users'));
  }
}
// resources/views/users/index.blade.php
@extends('layouts.app')

@section('content')
  <h1>Daftar Pengguna</h1>
  <ul>
    @foreach($users as $user)
      <li>{{ $user }}</li>
    @endforeach
  </ul>
@endsection
// routes/web.php
Route::get('/users', 'UserController@index');

Blade Templating

Blade adalah template engine Laravel yang menyediakan sintaks yang mudah digunakan untuk membuat tampilan yang dinamis.

// resources/views/layouts/app.blade.php
<!DOCTYPE html>
<html>
<head>
  <title>Aplikasi Saya</title>
</head>
<body>
  @yield('content')
</body>
</html>
// resources/views/home.blade.php
@extends('layouts.app')

@section('content')
  <h1>{{ $title }}</h1>
  <p>Selamat datang di aplikasi kami!</p>
@endsection

Fitur Blade:

  • @extends - Menggunakan layout
  • @section - Mendefinisikan section
  • @yield - Menampilkan section
  • {{ $variable }} - Menampilkan data (auto escape)
  • {!! $html !!} - Menampilkan HTML (tanpa escape)
  • @if, @else, @endif - Percabangan
  • @foreach, @endforeach - Perulangan
  • @include - Menyertakan view lain

Menjalankan Server

Laravel menyediakan server development bawaan yang bisa digunakan untuk pengembangan lokal.

# Jalankan server development
php artisan serve

# Dengan port tertentu
php artisan serve --port=8000

# Dengan host tertentu
php artisan serve --host=0.0.0.0

Perintah Artisan Berguna:

  • php artisan make:controller - Membuat controller baru
  • php artisan make:model - Membuat model baru
  • php artisan make:migration - Membuat migrasi database
  • php artisan migrate - Menjalankan migrasi
  • php artisan route:list - Melihat daftar route
  • php artisan tinker - Interaktif shell Laravel

Database dan Model

Laravel menggunakan Eloquent ORM untuk berinteraksi dengan database. Eloquent menyediakan sintaks yang mudah untuk operasi database.

# Membuat migrasi baru
php artisan make:migration create_users_table
// database/migrations/xxxx_create_users_table.php
public function up()
{
  Schema::create('users', function (Blueprint $table) {
    $table->id();
    $table->string('name');
    $table->string('email')->unique();
    $table->string('password');
    $table->timestamps();
  });
}
# Menjalankan migrasi
php artisan migrate
# Membuat model
php artisan make:model User
// app/Models/User.php
namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class User extends Model
{
  protected $fillable = ['name', 'email', 'password'];
}

Operasi Database dengan Eloquent:

  • User::all() - Mengambil semua data
  • User::find(1) - Mencari berdasarkan ID
  • User::where('active', 1)->get() - Query dengan kondisi
  • $user = new User(); $user->save() - Membuat data baru
  • $user->update() - Memperbarui data
  • $user->delete() - Menghapus data

Contoh Sederhana CRUD

Berikut contoh implementasi CRUD (Create, Read, Update, Delete) sederhana dengan Laravel.

// routes/web.php
Route::resource('posts', 'PostController');
# Membuat controller resource
php artisan make:controller PostController --resource
// app/Http/Controllers/PostController.php
public function index()
{
  $posts = Post::all();
  return view('posts.index', compact('posts'));
}

public function create()
{
  return view('posts.create');
}

public function store(Request $request)
{
  Post::create($request->all());
  return redirect()->route('posts.index');
}

public function show(Post $post)
{
  return view('posts.show', compact('post'));
}

public function edit(Post $post)
{
  return view('posts.edit', compact('post'));
}

public function update(Request $request, Post $post)
{
  $post->update($request->all());
  return redirect()->route('posts.index');
}

public function destroy(Post $post)
{
  $post->delete();
  return redirect()->route('posts.index');
}

Route Resource:

Route::resource() akan membuat 7 route standar untuk operasi CRUD:

  • GET /posts - posts.index
  • GET /posts/create - posts.create
  • POST /posts - posts.store
  • GET /posts/{post} - posts.show
  • GET /posts/{post}/edit - posts.edit
  • PUT/PATCH /posts/{post} - posts.update
  • DELETE /posts/{post} - posts.destroy

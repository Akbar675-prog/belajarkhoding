HTML

Struktur dasar dari semua halaman web. Pelajari elemen, tag, atribut, dan cara membangun halaman.

Apa itu HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML mendeskripsikan struktur halaman web menggunakan elemen-elemen yang ditandai dengan tag.

HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang digunakan untuk memberi struktur pada konten web.

Struktur Dasar Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar yang terdiri dari beberapa bagian penting:

<!DOCTYPE html> // Deklarasi tipe dokumen
<html lang="id"> // Elemen root dengan atribut bahasa
  <head> // Informasi meta tentang dokumen
    <title>Judul Halaman</title>
    <meta charset="UTF-8">
  </head>
  <body> // Konten yang ditampilkan di browser
    <!-- Konten halaman di sini -->
  </body>
</html>

Tag-Tag HTML Umum

1. Tag Dasar

Tag Fungsi Contoh
<html> Elemen root dokumen HTML <html>...</html>
<head> Berisi metadata dokumen <head>...</head>
<body> Berisi konten halaman <body>...</body>

2. Tag Konten

Tag Fungsi Contoh
<h1> - <h6> Heading level 1-6 <h1>Judul</h1>
<p> Paragraf teks <p>Paragraf...</p>
<a> Link/Anchor <a href="...">Link</a>
<img> Gambar <img src="..." alt="...">
<div> Pembungkus blok <div>...</div>
<span> Pembungkus inline <span>...</span>

Atribut Umum HTML

Atribut memberikan informasi tambahan tentang elemen HTML dan biasanya muncul dalam tag pembuka.

Atribut Fungsi Contoh
id Pengidentifikasi unik untuk elemen <div id="header">
class Nama kelas untuk styling/seleksi <p class="intro">
href URL tujuan untuk link <a href="...">
src Sumber untuk gambar/script <img src="...">
alt Teks alternatif untuk gambar <img alt="...">

Contoh Kode HTML Sederhana

Berikut contoh lengkap halaman HTML sederhana yang bisa Anda salin dan coba:

<!DOCTYPE html>
<html lang="id">
<head>
  <title>Halaman Pertamaku</title>
  <meta charset="UTF-8">
</head>
<body>
  <h1>Selamat Datang!</h1>
  <p>Ini adalah halaman HTML pertamaku.</p>
  <img src="gambar.jpg" alt="Deskripsi gambar">
  <a href="https://contoh.com">Kunjungi Contoh.com</a>
</body>
</html>

Salin kode di atas ke file dengan ekstensi .html dan buka di browser untuk melihat hasilnya.

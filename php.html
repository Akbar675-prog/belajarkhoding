PHP

Bahasa backend populer untuk membuat website dinamis. Pelajari sintaks dasar, variabel, form, dan koneksi database MySQL.

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa scripting server-side yang dirancang khusus untuk pengembangan web. PHP dijalankan di server dan menghasilkan HTML yang dikirim ke browser pengguna.

PHP adalah salah satu bahasa backend paling populer karena:

  • Mudah dipelajari: Sintaks yang sederhana untuk pemula
  • Kompatibilitas tinggi: Bekerja dengan hampir semua server dan database
  • Komunitas besar: Banyak dokumentasi dan dukungan
  • Frameworks populer: Seperti Laravel, CodeIgniter, Symfony
  • Open source: Gratis digunakan dan dikembangkan

PHP biasanya digunakan untuk:

  • Membuat halaman web dinamis
  • Mengolah data form
  • Mengakses database
  • Membuat session dan cookies
  • Membangun CMS seperti WordPress

Sintaks Dasar PHP

Kode PHP diawali dengan <?php dan diakhiri dengan ?>. File PHP harus disimpan dengan ekstensi .php.

<!DOCTYPE html>
<html>
<head>
  <title>Contoh PHP</title>
</head>
<body>
  <?php
  echo "Halo, Dunia!";
  ?>
</body>
</html>

Fitur dasar PHP:

  • Kode PHP dapat disisipkan dalam HTML
  • Setiap perintah diakhiri dengan titik koma (;)
  • Komentar menggunakan // untuk satu baris atau /* ... */ untuk banyak baris
  • Output ke browser menggunakan echo atau print

Variabel dan Tipe Data

Variabel dalam PHP diawali dengan tanda dollar ($) dan tidak perlu dideklarasikan tipe datanya (loosely typed).

<?php
// Deklarasi variabel
$nama = "Budi Santoso"; // String
$umur = 25; // Integer
$tinggi = 1.75; // Float
$isMahasiswa = true; // Boolean
$hobi = ["Membaca", "Bersepeda", "Memasak"]; // Array

// Menggabungkan string
echo "Nama: " . $nama . ", Umur: " . $umur;
?>

Tipe data utama dalam PHP:

  • String: Teks (contoh: "Hello")
  • Integer: Bilangan bulat (contoh: 42)
  • Float: Bilangan desimal (contoh: 3.14)
  • Boolean: True atau false
  • Array: Kumpulan nilai
  • Object: Instance dari class
  • NULL: Variabel tanpa nilai

Struktur Kontrol

1. Kondisi (if, else, switch)

<?php
$nilai = 85;

// If-else
if ($nilai >= 80) {
  echo "Nilai A";
} elseif ($nilai >= 70) {
  echo "Nilai B";
} else {
  echo "Nilai C";
}

// Switch-case
switch ($nilai) {
  case 100:
    echo "Sempurna!";
    break;
  case 90:
    echo "Bagus!";
    break;
  default:
    echo "Lumayan";
}
?>

2. Perulangan (loop)

<?php
// For loop
for ($i = 1; $i <= 5; $i++) {
  echo "Nomor: " . $i . "<br>";
}

// While loop
$x = 1;
while ($x <= 5) {
  echo "Nomor: " . $x . "<br>";
  $x++;
}

// Foreach (untuk array)
$buah = ["Apel", "Jeruk", "Mangga"];
foreach ($buah as $b) {
  echo "Buah: " . $b . "<br>";
}
?>

Form HTML dengan PHP

PHP dapat menerima data dari form HTML menggunakan variabel superglobal $_GET atau $_POST.

Contoh Form Login

<!-- form.html -->
<form action="proses.php" method="post">
  <input type="text" name="username">
  <input type="password" name="password">
  <button type="submit">Login</button>
</form>

// proses.php
<?php
$username = $_POST['username'];
$password = $_POST['password'];

if ($username == 'admin' && $password == '12345') {
  echo "Login berhasil!";
} else {
  echo "Username/password salah!";
}
?>

Perbedaan $_GET dan $_POST:

  • GET: Data dikirim melalui URL (terlihat di address bar), cocok untuk data tidak sensitif
  • POST: Data dikirim secara tersembunyi, cocok untuk data sensitif seperti password
  • Gunakan htmlspecialchars() untuk mencegah XSS (Cross-Site Scripting)

Koneksi Database MySQL

PHP dapat terhubung ke database MySQL menggunakan ekstensi MySQLi (MySQL Improved) atau PDO (PHP Data Objects).

<?php
// Koneksi ke database
$host = "localhost"; // Nama server
$user = "root"; // Username database
$pass = ""; // Password database
$db = "belajar_php"; // Nama database

// Membuat koneksi
$conn = mysqli_connect($host, $user, $pass, $db);

// Cek koneksi
if (!$conn) {
  die("Koneksi gagal: " . mysqli_connect_error());
}
echo "Koneksi berhasil!";

// Query SQL
$sql = "SELECT * FROM users";
$result = mysqli_query($conn, $sql);

// Tampilkan hasil
if (mysqli_num_rows($result) > 0) {
  while($row = mysqli_fetch_assoc($result)) {
    echo "ID: " . $row["id"] . ", Nama: " . $row["nama"] . "<br>";
  }
} else {
  echo "Tidak ada data";
}

// Tutup koneksi
mysqli_close($conn);
?>

Praktik terbaik koneksi database:

  • Gunakan prepared statements untuk mencegah SQL injection
  • Simpan informasi koneksi di file terpisah (config.php)
  • Selalu tutup koneksi setelah digunakan
  • Gunakan try-catch untuk menangani error
  • Hash password sebelum disimpan di database

XAMPP dan Localhost

Untuk menjalankan PHP di komputer lokal, Anda membutuhkan server seperti XAMPP, WAMP, atau LAMP.

XAMPP

XAMPP adalah paket server web yang berisi:

  • Apache (web server)
  • MySQL/MariaDB (database)
  • PHP dan Perl
  • phpMyAdmin (untuk mengelola database)

Download XAMPP

Cara Menggunakan

  1. Install XAMPP
  2. Jalankan Apache dan MySQL di XAMPP Control Panel
  3. Simpan file PHP di folder htdocs
  4. Akses via browser: http://localhost/nama-file.php
  5. Untuk phpMyAdmin: http://localhost/phpmyadmin

Struktur Folder XAMPP

xampp/
├── htdocs/ # Tempat menyimpan file PHP/HTML
│ ├── project1/ # Folder proyek pertama
│ └── index.php # File utama
├── php/ # File PHP
├── mysql/ # Database MySQL
└── ...

Contoh Proyek PHP

Berikut contoh sederhana aplikasi buku tamu dengan PHP dan MySQL:

// config.php
<?php
$host = "localhost";
$user = "root";
$pass = "";
$db = "buku_tamu";

$conn = mysqli_connect($host, $user, $pass, $db);
if (!$conn) {
  die("Koneksi gagal: " . mysqli_connect_error());
}
?>
// index.php
<?php
require 'config.php';

// Proses form
if (isset($_POST['submit'])) {
  $nama = $_POST['nama'];
  $email = $_POST['email'];
  $pesan = $_POST['pesan'];

  $sql = "INSERT INTO tamu (nama, email, pesan) VALUES (?, ?, ?)";
  $stmt = mysqli_prepare($conn, $sql);
  mysqli_stmt_bind_param($stmt, "sss", $nama, $email, $pesan);
  mysqli_stmt_execute($stmt);
}

// Ambil data tamu
$sql = "SELECT * FROM tamu ORDER BY id DESC";
$result = mysqli_query($conn, $sql);
?>

<!DOCTYPE html>
<html>
<body>
  <h1>Buku Tamu</h1>
  <form method="post">
    <input type="text" name="nama" placeholder="Nama">
    <input type="email" name="email" placeholder="Email">
    <textarea name="pesan" placeholder="Pesan"></textarea>
    <button type="submit" name="submit">Kirim</button>
  </form>

  <h2>Daftar Tamu</h2>
  <?php while($row = mysqli_fetch_assoc($result)): ?>
    <div>
      <h3><?= $row['nama'] ?></h3>
      <p><?= $row['pesan'] ?></p>
      <small><?= $row['email'] ?></small>
    </div>
  <?php endwhile; ?>
</body>
</html>
